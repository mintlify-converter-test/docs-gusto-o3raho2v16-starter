---
title: "Contractor Payment Webhooks"
description: "Events for contractor payments"
---
When subscribed to the `ContractorPayment` webhooks you will receive the following events:

| event\_type | description |
| --- | --- |
| contractor\_payment.created | Contractor has been created |
| contractor\_payment.cancelled | Contractor has been updated |

### Example payload

```json
{
  "event_type": "contractor_payment.created",
  "resource_type": "Company",
  "resource_id": 7757616924740529,
  "resource_uuid": "88f7cca1-dcad-4d20-84db-7fb80303d69f",
  "entity_type": "ContractorPayment",
  "entity_id": 7757491335866020,
  "entity_uuid": "7b1d0df1-6403-4a06-8768-c1dd7d24d27a",
  "timestamp": 1658773140
}
```
